# Portfolio Guillaume Perrin

## Structure des fichiers

```
portfolio-photo/
├── index.html          ← site public
├── data.json           ← albums, diaporama, réglages
├── netlify.toml        ← config Netlify
├── admin/
│   ├── index.html      ← interface Netlify CMS
│   └── config.yml      ← config CMS
└── photos/
    ├── diaporama/      ← photos du slideshow
    └── albums/         ← photos par album
```

## Déploiement sur Netlify

1. Uploadez ce dossier sur GitHub (dépôt : portfolio-photo)
2. Connectez Netlify à ce dépôt
3. Netlify déploie automatiquement
4. Activez Netlify Identity dans le dashboard Netlify
5. Invitez-vous par email
6. Accédez à /admin pour gérer vos photos
